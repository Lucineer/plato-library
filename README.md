# PLATO Library — Knowledge Base Room

Submit, query, and rate knowledge entries.

## Actions
- `submit` — title, content, category, tags, source
- `query` — filter by category, tags, author (sorted by rating)
- `rate` — 1-5 score on entries

## Categories
Auto-created on first submission. Suggested: `ptx-lessons`, `cuda-patterns`, `constraint-laws`, `fleet-protocols`, `esp32-notes`
